About this Website

Waypoint helps you find nearby placed using your location or with a given city name. You can choose from a variety of categories to filter your search and choose the number of results you would like to see. Additionally you can see the weather forecast for each of the places you find. Please visit the contact us to learn more info about the website developers.

Waypoint was summer project done by two college students who are studying in computer-related majors. It was created to help hese two students gain more experience in working with code, APIs, and web development. To learn more about the creators visit the contact us page (link here)

This website was made posssible through the use of APIs. To gather data about places near a certain location we used the HERE Maps API. This API enables us to help search for places using location information such as a city name or latitude and longitude coordinates. We were also able to filter through places using specific categories and that were provided by HERE maps. This Website allows the user use their current location to searches for places nearby. This was done with the help of the Abstract API. The Abstract API tracks the IP location of the user when they search for places nearby and provides us with latitude and longitude coordinates of the user's location to input into the HERE maps API. Additionally for each place provided when a user performs a search the current weather information is displayed. This was done with the use of the OpenWeatherMap API. They were able to give weather information such as the temperature and weather conditions based on latiude and logitude coordinates.
